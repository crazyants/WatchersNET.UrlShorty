To get the Module running you need Access to the IIS Manager!

  1. Open Internet Service Manager - From your server's desktop, launch the Internet Services Manager (usually located at Start->Programs->Administrative Tools->Internet Services Manager)
  2. Click the + to the left of the server name
  3. Right-click on "DotNetNuke" (or whatever you may have renamed your DNN App), and click on "Properties"
  4. Locate the Custom Errors Tab - Click on the Custom Errors tab (See Image below)
  5. Click on the 404 HTTP Error then click "Edit Properties".
  6. Use Message Type URL and Enter /DesktopModules/WatchersNET.UrlShorty/Redirect.aspx as URL and click OK. Keep clicking OK to dismiss the windows, then close the IIS window.
